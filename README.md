**🎬 BB Scene Director**

Расширение для моментального управления стилем, атмосферой и эмоциональным настроем, без необходимости каждый раз редактировать ваш пресет.

**Что внутри?**

  * **🎚 Эквалайзер стилей:** Добавляйте любые стили (Экшен, Драма, Жестокость, Романтика) и настраивайте их интенсивность ползунками от 0 до 10. Нейросеть мгновенно поймет, на чём сделать акцент.
  * **⚙️ Два режима работы (Автоинъекция или макрос):** По умолчанию плагин сам аккуратно вставляет инструкции перед генерацией сообщения. Но вы можете включить режим макроса и вписать `{{bb_scene}}` в любое место вашего пресета.
  * **🕹 Удобный HUD:** Выдвижная боковая панель с предпросмотром текущей инструкции. Включайте, выключайте и удаляйте стили в один клик.

**⚙️ Установка:**

1.  Откройте SillyTavern.
2.  Перейдите во вкладку *Расширения -\> Установить расширение*.
3.  Вставьте ссылку на этот GitHub репозиторий: `https://github.com/maxkara14/BB-Scene-Director`
4.  Перезагрузите страницу (F5).
5.  Нажмите на иконку хлопушки (🎬) с левой стороны экрана в любом активном чате, чтобы открыть панель режиссёра.
